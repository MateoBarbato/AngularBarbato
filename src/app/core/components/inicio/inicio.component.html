Bienvenido! Como usar:

Para navegar en la pagina web se puede utilizar los links en la barra de navegacion para un acceso rapido y sencillo.

Estos links poseen Guards que restringen la navegacion si uno no esta logueado.

Para poder acceder a cursos y alumnos se puede utilizar este usuario:

usuario: user || contraseña: user

Si se quiere acceder a traves de la url tambien se puede , siendo las rutas estas:

  • /cursos Te dirige directamente al listado de cursos
  • /alumnos Te dirige directamente al listado de cursos

En la parte superior izquierda de esta card aparece un boton, el cual acciona el slide de la navegacion lateral.

Ahi se pueden encontrar acciones para agregar nuevos usarios a la base de datos para testear las funcionalidades

Durante el periodo de prueba de la pagina estara habilitado el usuario siguiente:

usuario: admin || contraseña: admin

Tener en cuenta que para poder editar y agregar cursos o alumnos es necesario tener la autenticacion de admin
